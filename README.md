## หลักการ OOP ที่ใช้
- **Encapsulation**: ซ่อนข้อมูลภายใน `private`  
- **Abstraction**: ใช้ `List<T>` และเมธอดเพื่อจัดการข้อมูล  
- **Inheritance**: `Student` และ `Advisor` สืบทอดจาก `Person`  
- **Polymorphism**: Override `GetInfo()`
